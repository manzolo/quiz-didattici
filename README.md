## 📚 Quiz Disponibili

### Matematica

#### 📏 Quiz Equivalenze
Esercitati con le conversioni tra unità di misura:
- **Massa**: kg, hg, dag, g, dg, cg, mg
- **Lunghezza**: km, hm, dam, m, dm, cm, mm
- **Volume**: kl, hl, dal, l, dl, cl, ml

Funzionalità:
- 10 domande per sessione
- Valori casuali e variati
- Supporto per moltiplicazioni e divisioni
- Filtro per categoria (tutte, solo massa, solo lunghezza, solo volume)
- Storico degli ultimi 10 voti con trend
- Tooltip di aiuto con scala di conversione

#### 🧮 Calcolatore di Equivalenze
Converti istantaneamente tra diverse unità di misura:
- Inserisci un valore in qualsiasi unità
- Visualizza automaticamente tutte le conversioni
- Categoria casuale all'avvio per varietà

## 🚀 Come Usare

### 🌐 Online (GitHub Pages)
Il modo più semplice per usare i quiz:

1. Vai su **[https://manzolo.github.io/quiz-didattici/](https://manzolo.github.io/quiz-didattici/)**
2. Scegli la materia desiderata
3. Inizia a esercitarti!

Nessuna installazione necessaria, funziona direttamente nel browser!

### 💻 Locale
```bash
# Clona il repository
git clone https://github.com/manzolo/quiz-didattici.git

# Naviga nella cartella
cd quiz-didattici

# Apri index.html con il tuo browser preferito
firefox index.html
# oppure
google-chrome index.html
# oppure
open index.html  # su macOS
```

## 🚀 Deploy su GitHub Pages

Vuoi hostare la tua versione su GitHub Pages?

1. Fai un fork del repository
2. Vai su **Settings** → **Pages** nel tuo fork
3. Sotto "Source":
   - Branch: `main`
   - Folder: `/ (root)`
4. Click su **Save**
5. Dopo 1-2 minuti, il sito sarà disponibile su `https://[tuo-username].github.io/quiz-didattici/`
