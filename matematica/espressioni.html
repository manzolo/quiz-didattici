Regole per le Espressioni

Ordine delle parentesi
Calcola dall'interno verso l'esterno:
prima  ( )  poi  [ ]  poi  { }
Numeri relativi
Addizione: segni uguali → somma  |  segni diversi → differenza
(+3) + (-5) = -2    (-3) + (-4) = -7
Moltiplicazione: segni uguali → +  |  segni diversi → −
(-3) × (-4) = +12    (-3) × 4 = -12
Potenze con base negativa
Esponente pari → risultato sempre positivo
(-3)² = +9    (-5)² = +25
Esponente dispari → mantiene il segno della base
(-2)³ = -8    (-1)³ = -1
Priorità delle operazioni
Dentro le parentesi: prima potenze, poi × e ÷, poi + e −
2 + 3 × (-4) = 2 + (-12) = -10
