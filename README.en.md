- 🌍 **Multilingual**: Interface available in Italian and English
- 📱 **Responsive**: Works perfectly on desktop, tablet, and smartphone
- 💾 **Offline First**: No internet connection required
- 📊 **Progress Tracking**: Results history saved locally
- 🎨 **Modern Design**: Colorful and attractive interface
- ♿ **Accessible**: Design meant to be usable by everyone

## 📚 Available Quizzes

### Mathematics

#### 📏 Unit Conversion Quiz
Practice conversions between units of measurement:
- **Mass**: kg, hg, dag, g, dg, cg, mg
- **Length**: km, hm, dam, m, dm, cm, mm
- **Volume**: kl, hl, dal, l, dl, cl, ml

Features:
- 10 questions per session
- Random and varied values
- Support for multiplication and division
- Category filter (all, mass only, length only, volume only)
- History of last 10 grades with trend
- Help tooltip with conversion scale

#### 🧮 Unit Converter Calculator
Instantly convert between different units of measurement:
- Enter a value in any unit
- Automatically view all conversions
- Random category at startup for variety

## 🚀 How to Use

### 🌐 Online (GitHub Pages)
The easiest way to use the quizzes:

1. Go to **[https://manzolo.github.io/quiz-didattici/](https://manzolo.github.io/quiz-didattici/)**
2. Choose the desired subject
3. Start practicing!

No installation required, works directly in the browser!

### 💻 Local
```bash
# Clone the repository
git clone https://github.com/manzolo/quiz-didattici.git

# Navigate to the folder
cd quiz-didattici

# Open index.html with your preferred browser
firefox index.html
# or
google-chrome index.html
# or
open index.html  # on macOS
```

## 🚀 Deploy on GitHub Pages

Want to host your own version on GitHub Pages?

1. Fork the repository
2. Go to **Settings** → **Pages** in your fork
3. Under "Source":
   - Branch: `main`
   - Folder: `/ (root)`
4. Click **Save**
5. After 1-2 minutes, the site will be available at `https://[your-username].github.io/quiz-didattici/`
